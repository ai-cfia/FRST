FRST is a web-based application that supports the analysis and visualization of the expected cost, benefits, and uncertainty of a project, project scenarios, or portfolio of projects. FRST was built to assess projects that aim to re-align resources that are currently dedicated to an existing function, to a function that is deemed higher priority by the organization.

How it Works

There are three interfaces associated with the FRST: "Home", "Charts", and "Projects".

The "Home" Page

You are currently on the “Home” page. It provides an explanation of the features of FRST and provides examples of how to use the tool.

The "Charts" Page

The “Charts” tab illustrates projects cost and released benefit over the time period, project indicators, and project complexity and risk. You can change the project being displayed by selecting from the drop down list. You can also compare projects and project scenarios, as well as build a project portfolio.

The "Projects" Page

The application allows users to input new projects and project details through the “Projects” page.

What is “Released Benefit”?

The released benefit is the amount of resources no longer dedicated to the implementation of a specific program as a result of a modernization initiative.

What is “Cost”?

The cost is the amount of resources required to implement a modernization initiative from planning through to implementation that will result in released benefit.

What is an “FTE”?

FTE is the abbreviation of “full-time equivalent”. For the purposes of FRST, an FTE is equal to $100,000 annually.

What are “Operating Expense”?

Operating expense is operating cost or release that is incurred to conduct the existing activity, or in order to complete the realignment project.

What is “Risk and Complexity”?

Risk and complexity are the factors that will contribute to the uncertainty in the project achieving the estimated cost and released benefit.

How is Cost of Complexity and Risk Calculated?

The complexity and risk is based on the weighting of Treasury Board Secretariat’s “Project and Complexity Risk Assessment” and inflation factor in order to translate into the same unit of measure.

What can FRST do

Project Analysis

Once the user has input all project details, FRST provides some illustrative and analytical features in the “Charts” tab.

Cost / Released Benefit

The Cost and Released Benefit is a chart that displays the estimated cost and released benefit per quarter over the time horizon estimated.

Maximum Net Benefit

The maximum net benefit is the result of subtracting the estimated cost from the released benefit without factoring in the potential uncertainty.

Risk and Complexity

The Risk and Complexity chart illustrates the areas where the higher relative risks are according to the project.

Minimum Net Benefit

The minimum net benefit is the result of subtracting the estimated cost and released benefit while factoring in the impact of potential uncertainty.

Project Scenario Analysis

Users can input different iterations of a similar project (i.e. higher resource levels, or different risk contexts) and compare the two projects using the drop down menu.
